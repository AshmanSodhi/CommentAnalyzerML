# CommentAnalyzerML

## Objective
  + To analyze the given comment or statement and tell whether it is toxic/severe toxic/obscene threat/insult/identity hate

## Usage
  + Can be used to analyze comments under any social media posts and to filter them

## Dependencies Used
  + Tensorflow
  + Numpy
  + Pandas
